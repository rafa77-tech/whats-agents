# Epic 05 - Admin/Qualidade UI

## Objetivo
Expor avaliacao de conversas e sugestoes de prompt no dashboard.

## Stories

### S43.E5.1 - Lista de Conversas para Avaliacao
**Objetivo:** Mostrar conversas com filtros e status de avaliacao.

**Tarefas**
1. Pagina de avaliacao com filtros.
2. Badge de avaliada/pendente.

### S43.E5.2 - Detalhe da Conversa com Avaliacao
**Objetivo:** Permitir avaliar conversa e registrar notas.

**Tarefas**
1. Form de avaliacao (naturalidade, persona, objetivo).
2. Salvar e atualizar status.

### S43.E5.3 - Sugestoes de Prompt
**Objetivo:** Listar e revisar sugestoes.

**Tarefas**
1. Listar sugestoes pendentes.
2. Acoes de aprovar/rejeitar.
