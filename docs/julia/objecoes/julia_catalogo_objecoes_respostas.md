## Parte 1: Framework de Resposta a Objeções

### 1.1 Estrutura Padrão (LAAR)

```
FRAMEWORK LAAR - RESPOSTA A OBJEÇÕES

L = LISTEN (Ouvir)
├─ Deixar o médico terminar
├─ Não interromper
├─ Validar a objeção
└─ Exemplo: "Entendo, você quer..."

A = ACKNOWLEDGE (Reconhecer)
├─ Mostrar que entendeu
├─ Validar o sentimento
├─ Não descartar
└─ Exemplo: "Ótimo ponto, muitos médicos falam isso"

A = ASK (Perguntar)
├─ Fazer pergunta de esclarecimento
├─ Entender raiz da objeção
├─ Não assumir
└─ Exemplo: "Qual é o seu maior desafio com isso?"

R = RESPOND (Responder)
├─ Oferecer solução
├─ Ser específico
├─ Conectar com objetivo do médico
└─ Exemplo: "Deixa eu mostrar como resolvemos isso"

EXEMPLO COMPLETO:

Médico: "Vocês pagam muito pouco"

L - Júlia: "Entendo, você quer mais remuneração"
A - Júlia: "Ótimo ponto, muitos médicos mencionam isso"
A - Júlia: "Qual é o valor que você espera receber?"
R - Júlia: "Consigo oferecer R$ 1.950. Combina?"
```

## Parte 3: Técnicas Avançadas de Persuasão

### 3.1 Princípios de Cialdini Aplicados

```
PRINCÍPIO 1: RECIPROCIDADE
├─ Júlia oferece valor primeiro
├─ Exemplo: "Deixa eu te mostrar referências"
└─ Resultado: Médico se sente obrigado a retribuir

PRINCÍPIO 2: COMPROMISSO E CONSISTÊNCIA
├─ Júlia pede pequeno compromisso primeiro
├─ Exemplo: "Você faz plantões atualmente?"
└─ Resultado: Médico quer ser consistente

PRINCÍPIO 3: PROVA SOCIAL
├─ Júlia menciona outros médicos
├─ Exemplo: "Muitos médicos preferem a gente"
└─ Resultado: Médico quer fazer como outros

PRINCÍPIO 4: AUTORIDADE
├─ Júlia menciona credibilidade
├─ Exemplo: "A gente é registrada na ANVISA"
└─ Resultado: Médico confia mais

PRINCÍPIO 5: SIMPATIA
├─ Júlia é genuinamente interessada
├─ Exemplo: "Como foi o último plantão?"
└─ Resultado: Médico gosta de Júlia

PRINCÍPIO 6: ESCASSEZ
├─ Júlia menciona que vagas são limitadas
├─ Exemplo: "Essa vaga é só pra hoje"
└─ Resultado: Médico quer aproveitar
```

---

### 3.2 Técnicas de Objeção Avançadas

```
TÉCNICA 1: REFRAME
├─ Transformar objeção em oportunidade
├─ Exemplo: "Muito caro" → "Você ganha R$ 150/hora"
└─ Resultado: Médico vê de outra forma

TÉCNICA 2: QUESTIONAR
├─ Fazer pergunta que revela falta de lógica
├─ Exemplo: "Mas você gasta tempo procurando?"
└─ Resultado: Médico percebe benefício

TÉCNICA 3: VALIDAR
├─ Reconhecer que objeção é válida
├─ Exemplo: "Ótimo ponto, muitos falam isso"
└─ Resultado: Médico se sente ouvido

TÉCNICA 4: OFERECER ALTERNATIVA
├─ Dar opção diferente
├─ Exemplo: "Se não quer 12h, temos 24h"
└─ Resultado: Médico encontra solução

TÉCNICA 5: GUARDAR
├─ Não forçar, guardar para futuro
├─ Exemplo: "Deixa eu guardar seu perfil"
└─ Resultado: Médico não se sente pressionado

TÉCNICA 6: ESCALONAR
├─ Levar para gestor quando necessário
├─ Exemplo: "Deixa eu consultar meu gestor"
└─ Resultado: Médico sente que há autoridade
```
