# Design System (tokens e componentes)

Base: manual Jull.ia + estilo minimalista data-first.

## Tokens

### Cores
- primary: #ff1200
- accent: #ff0080
- support: #972627
- alert: #e00319
- black: #000000
- gray: #bababa
- white: #ffffff

Sugestao de neutros adicionais (para UI):
- gray-100: #f7f7f7
- gray-200: #eeeeee
- gray-300: #dddddd
- gray-700: #4a4a4a

### Tipografia
- display: Fredoka Medium
- display-light: Fredoka Light
- body: Barlow Regular

Tamanhos base (px):
- h1: 28
- h2: 22
- h3: 18
- body: 14
- small: 12

### Spacing
- xs: 4
- sm: 8
- md: 16
- lg: 24
- xl: 32
- xxl: 48

### Bordas
- radius-sm: 6
- radius-md: 10
- radius-lg: 14
- border: 1px solid #eeeeee

### Sombra
- shadow-sm: 0 1px 2px rgba(0,0,0,0.06)
- shadow-md: 0 6px 16px rgba(0,0,0,0.08)

## Componentes

### Botao
- Primary: fundo #ff1200, texto branco
- Secondary: borda #dddddd, texto #000000
- Danger: fundo #e00319, texto branco

### Badge/Status
- Sucesso: texto #0f8a4b, fundo #e9f8ef
- Alerta: texto #b54800, fundo #fff4e5
- Erro: texto #c80021, fundo #ffe8eb
- Neutro: texto #555, fundo #f1f1f1

### Card
- Fundo branco, borda #eeeeee, radius-md
- Header opcional, spacing md

### Tabela
- Header leve (fundo #f7f7f7)
- Linhas com hover sutil
- Coluna de acoes sempre na direita

### Input
- Altura 36px, radius-sm, borda #dddddd
- Focus: borda #ff1200, shadow-sm

## Layout
- Grid 12 colunas
- Max width conteudo: 1200-1280px
- Sidebar fixa 240px (colapsavel 72px)
